# nahida

让智慧之神指引你吧。

本项目由两部分组成

`realtime_damage_shower`：在游戏中触发战斗时，能自动实时显示当前队伍角色的伤害占比和全队DPS。

`Nahida`：自动记录战斗期间的收发数据包，结束后统一分析，得到包含武器，角色面板，dps及变化情况，能量变化等信息的图片。

两部分互相独立，以满足不同人群的需要。
